== Installation ==
1. Upload "bookin60.zip" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. You first need to add Services, Frequency, Pricing parameters, Add-ons in order to make the booking work after adding shortcode to page.
